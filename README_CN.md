# 🎮 Crypto Guy - 加密货币模拟交易游戏

[English README](./README.md)

像素风格的加密货币模拟交易游戏，以 $1,000 起步，在疯狂的加密市场中活下去！

## 🚀 功能特性

- 💰 初始资金 $1,000
- ₿ BTC 交易（买入/卖出/持仓）
- 🎲 随机事件系统影响市场走势
- 📅 月度结算机制
- 📱 移动端适配
- 🕹️ 复古像素风 UI

## 🛠️ 技术栈

- React 18 + TypeScript
- Vite
- Zustand（状态管理）
- TailwindCSS
- Press Start 2P 字体

## 📦 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 🌐 部署

推荐平台：
- Vercel
- Netlify
- Cloudflare Pages

连接 GitHub 仓库即可自动构建部署。

## 🎯 游戏玩法

1. 阅读当月事件
2. 根据事件判断买入、卖出或持仓
3. 点击"结束当月"查看结算
4. 资金归零则游戏结束

**目标：** 在加密市场中尽可能活得久！
